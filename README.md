## Overview

This is a multi-module umbrella project for [Jackson](../../../jackson)
standard binary dataformat backends.

Currently included are:

* [Avro](avro/)
* [CBOR](cbor/)
* [Protobuf](protobuf/)
* [Smile](smile/)
